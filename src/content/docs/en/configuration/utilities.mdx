Athena OS has several commands and applications that could be useful for its usage. In Athena OS they can be accessed by [Tool Recipe](/en/resources/athena-welcome/#tool-recipe) button. They can vary between Athena Arch and Athena Nix.

| Command | Description | Athena Arch | Athena Nix |
| ------- | ----------- | ---- | ---- |
| **armcord** | Cross-platform, all-in-one voice and text chat privacy-based Discord application. | X | X |
| **asciinema** | Record and share your terminal sessions. | X | X |
| **athena-theme-tweak** | Switch to different Athena themes. | X | |
| **athena-welcome** | Welcome application that allows main security tools installation and mirrorlist update. | X | X |
| **bashtop** | Resource monitor that shows usage and stats for processor, memory, disks, network and processes. | X | |
| **bat** | A **cat** clone with syntax highlighting and Git integration. | X | X |
| **bfetch** | SuperB general-purpose fetch displayer. | X | X |
| **bless** | High-quality, full-featured hex editor. | X | X |
| **btrfs-assistant** | GUI management tool to make managing a Btrfs filesystem easier. | X | |
| **chat-gpt** | ChatGPT Desktop application. | X | |
| **codium** |  Community-driven, freely-licensed binary distribution of VS Code. | X | X |
| **convert** | Convert between image formats as well as resize an image, blur, crop, despeckle, dither, and much more. | X | X |
| **downgrade** | Downgrade one (or multiple) packages by using the pacman cache or the Arch Rollback Machine. | X | |
| **git** | Fast, scalable, distributed revision control system. | X | X |
| **la** | An **lsd -a** alias. | X | X |
| **ll** | An **lsd -alFh** alias. | X | X |
| **lsd** | An **ls** command with a lot of pretty colors and some other stuff. | X | X |
| **nano** | Simple terminal-based text editor. | X | X |
| **ncdu** | Check which directories are using your disk space. | X | X |
| **neofetch** | CLI system information tool written in BASH. | X | X |
| **nvim** | Vim-fork focused on extensibility and usability. | X | X |
| **octopi** | A powerful Pacman (Package Manager) front end using Qt libs. | X | |

| **pacman** | Arch Linux package manager. | X | |
| **pactree** | Package dependency tree viewer. | X | |
| **paru** | Pacman wrapping AUR helper with lots of features and minimal interaction. | X | |

| **vim** | Highly configurable text editor built to make creating and changing any kind of text very efficient. | X | |
| **vnstat** | Console-based network traffic monitor. | X | X |
| **xcp** | An extended **cp** command. | X | X |
| **z** | A smarter **cd** command for your terminal. | X | X |
