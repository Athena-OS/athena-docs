In Athena OS, you can select among several themed environments for all tastes. These themes set several elements of your environment. A theme can be set by:
<Tabs>
<TabItem label="Athena Arch">
Athena Tweak Tool, just searching it in the application menu, or running
```
athena-tweak-tool
```
It is flexible application to tweak your Athena OS environment.

![image](https://github.com/user-attachments/assets/b90de29c-8b8c-4362-b766-01daeae7d757)

![image](https://github.com/user-attachments/assets/c53260f8-9b05-4ed3-afa1-02bcef584a16)

![image](https://github.com/user-attachments/assets/3a2481aa-c2bd-45b2-9c99-3c24f52e0e94)

</TabItem>
<TabItem label="Athena Nix">
editing `theme` variable in `/etc/nixos/configuration.nix` by typing the theme name you wish. Click [here](https://athenaos.org/en/configuration/nix-config/#theme) to see the list of valid theme name values.
</TabItem>
</Tabs>

Some themes have more wallpapers stored in `/usr/share/backgrounds`.
