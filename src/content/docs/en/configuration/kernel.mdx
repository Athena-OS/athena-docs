Athena OS adopts the [Zen Kernel](https://github.com/zen-kernel/zen-kernel/wiki/Detailed-Feature-List) because it is one of the officially supported kernels and it is characterized to be efficient and to have low-latency computing.

Major features of Zen Kernel can be read on [Liquorix](https://liquorix.net/).
